# 🎮 Grafikli Karakter Seçimli Web Oyunu

Bu proje basit bir **hesap sistemli karakter seçme web oyunudur.**

## 🚀 Özellikler

* 🔐 Giriş / Kayıt sistemi
* 🎮 Karakter seçimi
* 🧙‍♂️ Grafikli karakter kartları
* 🎯 Oyun başlangıç ekranı
* 💾 LocalStorage ile veri kaydetme

## 🖼️ Karakterler

* ⚔️ Savaşçı
* 🔮 Büyücü
* 🏹 Okçu

## 🛠️ Kullanılan Teknolojiler

* HTML
* CSS
* JavaScript

## 📂 Dosyalar

* index.html → Karakter seçimi
* login.html → Giriş / Kayıt
* game.html → Oyun sayfası
* style.css → Tasarım
* script.js → Kodlar

## ⚡ Kurulum

1. Repo indir
2. Dosyaları yükle
3. login.html aç
4. Hesap oluştur
5. Karakter seç

## 🎯 Amaç

Basit bir web tabanlı karakter seçme sistemi oluşturmak.
